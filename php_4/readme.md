# PHP_4

## Задача - подключить скрипт из другого файла

Подключение файлов и вывод сообщений в браузере:

- "Привет мир!" сохранить в переменной в файле index.php
- Подключить в index.php файл с разметкой документа layout.php
- Вставить "Привет мир!" в теле документа и показать в браузере

---
В папке с проектом php_4:

1. Написать базовую разметку документа в файле layout.php;

2. В разметке вывести значение переменной;

3. Запустить локальный сервер в терминале:

      ./server.bat

Перейти из терминала в браузер, кликнув левой кнопкой мыши по ссылке в строке:

    Server (http://localhost:4000) started

через нажатый Ctrl.

В открывшемся браузере должен загрузиться документ с тектсом

      "Привет мир!"
      Подключен из layout.php

Чтобы остановить сервер - нажать в терминале VS Code комбинацию Ctrl + C
