# Язык PHP

## Настройка для Windows

 1. Скачать PHP [официального сайта](https://windows.php.net/downloads/releases/php-8.5.1-nts-Win32-vs17-x64.zip).
 2. Создать папку на диске С:\php
 3. Распаковать содержимое архива в С:\php
 4. В переменную среды PATH добавить путь С:\php
 5. Установить редактор VS Code.
