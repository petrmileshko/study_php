# Язык PHP

## Настройка для Windows

 1. Установить редактор VS Code.
 2. Скачать PHP для ОС Windows [официального сайта](https://windows.php.net/downloads/releases/).
 3. Создать папку на диске С:\php
 4. Распаковать содержимое архива в С:\php
 5. В переменную среды PATH добавить путь С:\php
