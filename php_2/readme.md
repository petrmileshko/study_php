# PHP_2

## Задача

Вывод таблицы перевода температур по Фаренгейту в Цельсия, от 0 до 300 с шагом 20

---
Проверка программы:

    php index.php
    или
    php ./php_1/index.php

В консоле терминала VS Code должно появиться сообщение:

    ТАБЛИЦА ПЕРЕВОДА ТЕМПЕРАТУР

    Фаренгейт    Цельсия
    0                -17
    20                -6
            ...
    300              148
---
