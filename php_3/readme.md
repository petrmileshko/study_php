# PHP_3

## Задача - запустить локальный сервер и вывести в браузер строку

Вывод сообщения в браузере  - "Привет мир!"

---
В папке с проектом php_3:

1. Написать базовую разметку документа в файле index.php;

2. В разметке встроить код программы;

3. Создать файл server.bat и сохранить в нем:

      php -S localhost:4000

4. Запустить локальный сервер в терминале:

      server

Перейти из терминала в браузер, кликнув левой кнопкой мыши по ссылке в строке:

    Server (http://localhost:4000) started

через нажатый Ctrl.

В открывшемся браузере должно появиться сообщение - "Привет мир!"

Чтобы остановить сервер - нажать в терминале VS Code комбинацию Ctrl + C
